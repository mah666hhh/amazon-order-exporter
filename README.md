# アマゾン注文エクスポーター

Amazon.co.jpの注文履歴をCSV形式でエクスポートするChrome拡張機能

- **URL**: https://chrome.google.com/webstore/detail/iigehkabajngmjbnlnhmccocihkjknjj
- **ID**: iigehkabajngmjbnlnhmccocihkjknjj

## リンク

- **プライバシーポリシー**: https://mah666hhh.github.io/amazon-order-exporter/privacy-policy.html
- **GitHub**: https://github.com/mah666hhh/amazon-order-exporter

## 機能

- 注文履歴を年単位でエクスポート（2008年〜現在）
- 2つの出力形式：「注文単位」と「商品単位」
- 領収書・請求書リンクの取得に対応
- 複数ページの注文を自動で収集
